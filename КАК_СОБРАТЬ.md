# Как собрать проект (решение проблемы с путём на кириллице)

## Проблема
Если проект лежит в папке с кириллицей (например `c:\общая\bunker-online-app`), Node.js на Windows может неправильно обрабатывать путь, и сборка падает с ошибками:
- `Module not found` / `Can't resolve`
- `Couldn't find any pages or app directory`
- `ERR_PNPM_NO_SCRIPT Missing script: build`

## Решения

### 1. Переместить проект (рекомендуется)
Скопируйте проект в папку **без кириллицы**:
```
C:\projects\bunker-online-app
```
Затем в терминале:
```bash
cd C:\projects\bunker-online-app
pnpm install
pnpm run build
```

### 2. Запуск build.bat двойным щелчком
1. Откройте папку проекта в **Проводнике Windows**
2. Дважды щёлкните по файлу **build.bat**
3. Сборка запустится из корректного пути

### 3. Junction (если нужен путь без кириллицы без копирования)
Откройте **cmd от имени администратора** и выполните (путь замените на свой):
```cmd
mklink /J C:\bunker-build "c:\общая\bunker-online-app"
```
Затем:
```cmd
cd C:\bunker-build
pnpm install
pnpm run build
```

## Проверка
После успешной сборки появится папка `.next` и сообщение:
```
Build completed successfully
```
