# Характеристики в игре «Бункер»

Все возможные значения характеристик по категориям. Используются при генерации карточек игроков (кроме пола и возраста — см. ниже).

## Рандомизация

Для снижения повторений одних и тех же характеристик у одного игрока в разных играх:

1. **Криптостойкий выбор** — используется `crypto.randomInt` вместо `Math.random` для более равномерного распределения.
2. **История игрока** — при присвоении характеристик учитываются значения, которые игрок уже получал в прошлых играх. Предпочтение отдаётся тем, которые он давно не получал.
3. **Профессия** — выбор случайный среди свободных (раньше использовалась фиксированная последовательность по слоту).
